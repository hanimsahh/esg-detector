# ESG Detector

AI-powered platform for investigating the ESG pledges and plans 
of the world's leading sportswear brands.

## What it does

ESG Detector aggregates official sustainability reports, ESG-related 
news, and financial market data for Nike, Adidas and Puma into a 
single dashboard — allowing users to investigate ESG commitments 
without digging through hundreds of pages of annual reports.

## Features

- 📊 **Dashboard** — Key ESG metrics visualised via Looker Studio
- 📰 **ESG News** — Real-time news scraped and classified by E, S, G pillar
- 💹 **Financials** — Live financial data via Finnhub API
- 🤖 **Chatbot** — Ask questions about official ESG reports, powered by ChromaDB and Google Gemini

## Tech stack

Python · Streamlit · ChromaDB · Microsoft Phi-4 · Google Gemini · 
Finnhub API · BeautifulSoup · Looker Studio
